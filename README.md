# Task-13-BI-Dashboard-Storytelling

## Project Overview
This project is part of the **Data Analyst Internship – Task 13**.  
The goal of this task is to build an **executive-ready BI dashboard** using **Power BI**, focusing on KPI reporting and data storytelling.

The dashboard helps understand overall business performance through key metrics, trends, and comparisons.

---

## Dataset Used
**Global Superstore (Retail Sales Dataset)**

The dataset contains information about:
- Orders
- Sales
- Profit
- Categories
- Regions
- Products
- Order dates

---

## Tools Used
- Power BI Desktop

---

## KPIs Created
The following key performance indicators (KPIs) were created using DAX measures:
- **Total Sales**
- **Total Profit**
- **Profit Margin**

---

## Dashboard Features
The Power BI dashboard includes:

1. **KPI Cards**
   - Total Sales
   - Total Profit
   - Profit Margin

2. **Sales Trend Analysis**
   - Line chart showing Sales over time using Order Date

3. **Category-wise Sales**
   - Bar chart comparing sales across product categories

4. **Region-wise Performance**
   - Bar chart showing sales by region

5. **Top 10 Products**
   - Table displaying top 10 products based on total sales

6. **Interactive Slicers**
   - Region
   - Category
   - Order Date

7. **Insights Section**
   - Text box summarizing key business findings from the dashboard

---

## Key Insights
- Sales show a steady growth trend over time.
- Technology category contributes the highest sales.
- Certain regions perform better and generate higher revenue.
